# NovelReader

一个小说阅读器 Android 应用

## 项目简介

NovelReader 是一款功能完整的 Android 小说阅读器应用，提供流畅的阅读体验和便捷的小说管理功能。

## 主要功能

- 📖 流畅的小说阅读体验
- 📚 小说书库管理
- 🔖 书签和进度保存
- ⚙️ 阅读设置自定义
- 🌙 深色模式支持

## 技术栈

- **语言**: Java
- **平台**: Android
- **最低API级别**: 需要根据项目配置调整

## 快速开始

### 前置要求

- Android Studio
- JDK 8 或更高版本
- Android SDK

### 构建与运行

1. 克隆本仓库
```bash
git clone https://github.com/Zyy023577/NovelReader.git
```

2. 在 Android Studio 中打开项目

3. 连接 Android 设备或启动模拟器

4. 构建并运行应用
```bash
./gradlew build
./gradlew installDebug
```
